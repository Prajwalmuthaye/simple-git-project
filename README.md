# Version Control & DevOps Workflow Using Git and GitHub

# Project Overview

This project demonstrates the basic workflow of version control using Git and GitHub. It shows how developers manage source code, track changes, and collaborate in a DevOps environment.

The project contains a simple static webpage to understand repository creation, commits, branching, and pushing code to GitHub.

#Technologies Used

2. GitHub
3. HTML
4. CSS

# Git Workflow Used

Initialize repository
```
git init
```
Add project files
```
git add .
```
Commit changes
```
git commit -m "Initial commit"
```
Connect to GitHub repository
```
git remote add origin <repository-url>
```
Push code to GitHub

git push -u origin main
